This plugin will display posts in widget area as well as in post/page section with ticker when using shortcode [hm_recent_posts]. You need to set featured image in post section to display the thumbnail.

== Description ==

[WP Ticker Posts](http://www.hossnimubarak.com) works in two ways. One is a Widget which you can use in widget/sidebar area. Another is shortcoded by which you can display posts in Page section. You need to set featured image in post section to display the thumbnail. The post content is hidden in widget area or in mobile display, but shows in device which is larger than 767px. To display in post or page use the shortcode [hm_recent_posts]

== Installation ==

Install WP Ticker Posts as a regular WordPress plugin. Here are different ways to install plugin:

**Install by search plugin:**

1. In your Admin, go to menu Plugins > Add New
2. Search WP Ticker Posts
3. Click to install
4. Activate the plugin
5. Set the widget in Admin Widgets area.

**Install by upload plugin:**

1. Download the latest version of WP Ticker Posts (.zip file)
2. In your Admin, go to menu Plugins > Add New
3. Select "Upload Plugin"
4. Click on "Choose File"
5. Select downloaded hm-recent-posts-widget.zip and click on "Install Now" button
6. Activate the plugin
7. HM Recent Posts will appear in Admin Widget area.

== Frequently Asked Questions ==

= Post thumbnail not displaying =

You need to set featured image in post section to display the thumbnail.

= How to show in Widget/Sidebar? =

Go to Appearance > Widgets. Find HM Recent Posts. Drag and drop in your widget area.

= How to show in Post/Page? =

Just use the shortcode: [hm_recent_posts]
